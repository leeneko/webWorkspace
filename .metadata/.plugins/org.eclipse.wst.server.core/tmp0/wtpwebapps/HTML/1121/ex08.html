마크업[편집]
HTML 마크업은 HTML 요소(엘리먼트Elements)와 그들의 속성(Attributes)과 문자 기반
데이터 형태와 문자 참조와 엔티티 참조를 포함하는 몇 가지 핵심 구성 요소로 이루어져있다.
<!doctype html> <html> <head> <title>Hello HTML</title> </head> <body> <p>Hello World!</p> </body> </html>
Hello world 프로그램 프로그래밍 언어와 스크립트 언어 그리고 마크업 언어를 비교하기 위해
사용되는 일반적인 컴퓨터 프로그램이다. 그리고 HTML에서의 Hello world 프로그램은 단 9줄에 불과하다.네이버에서 확인하기
